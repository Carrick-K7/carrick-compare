# T-002: 行政区域数据结构设计

## 目标
定义行政区域和GDP数据的 TypeScript 类型，创建数据文件结构和加载工具函数。

## 详细要求

### 1. 类型定义文件 ✅

- `src/types/region.ts` - 区域类型定义
- `src/types/gdp.ts` - GDP数据类型定义  
- `src/types/comparison.ts` - 对比相关类型定义
- `src/types/index.ts` - 统一导出

### 2. 数据加载工具 ✅

`src/utils/dataLoader.ts` - 实现以下函数:
- `getRegionByCode(code)` - 根据代码获取区域
- `getRegionsByLevel(level)` - 按级别获取区域
- `getChildRegions(parentCode)` - 获取子区域
- `getRegionGDP(code, year, granularity)` - 获取GDP数据
- `getRegionHistory(code)` - 获取历史数据
- `searchRegionsByName(name)` - 按名称搜索 (额外功能)

### 3. 数据目录结构 ✅

```
src/data/
├── regions.json        # 行政区域基础数据（20个城市/省份）
├── gdp-annual.json     # 年度GDP数据 (2020-2024)
├── gdp-half.json       # 半年度GDP数据
└── index.ts            # 数据导出
```

### 4. 示例数据 ✅

- regions.json: 包含广东、深圳、北京、上海、杭州、苏州、南京、重庆、成都、武汉、青岛、郑州等示例区域
- gdp-annual.json: 2020-2024年示例GDP数据（55条记录）
- gdp-half.json: 2023-2024年半年度示例数据（14条记录）
