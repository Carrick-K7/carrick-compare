# T-001: 项目初始化与架构搭建

## 状态
✅ **已完成**

## 目标
创建 Carrick Compare 项目基础架构，配置 React + TypeScript + Vite 开发环境，集成 Tailwind CSS 和 shadcn/ui。

## 完成项

### 1. ✅ 创建 Vite 项目
- 使用 `react-ts` 模板初始化
- 项目路径: `/root/.openclaw/workspace/projects/carrick-compare`

### 2. ✅ 安装依赖
- tailwindcss, postcss, autoprefixer
- react-router-dom
- recharts
- lucide-react
- clsx, tailwind-merge
- @radix-ui/react-slot, @radix-ui/react-select
- class-variance-authority

### 3. ✅ 配置 Tailwind CSS
- 使用 Tailwind v4 CSS-first 配置
- 配置了 shadcn/ui 主题变量（primary: blue）
- 支持 CSS 变量主题系统

### 4. ✅ 初始化 shadcn/ui 组件
手动创建组件（适配 Tailwind v4）：
- button - 按钮组件
- input - 输入框组件
- card - 卡片组件
- select - 选择器组件

### 5. ✅ 创建目录结构
```
src/
├── components/
│   └── ui/          # shadcn 组件
├── data/            # 静态数据
├── types/           # TypeScript 类型
├── utils/           # 工具函数 (cn 函数)
├── pages/           # 页面组件
│   ├── HomePage.tsx
│   └── ResultPage.tsx
├── App.tsx
└── main.tsx
```

### 6. ✅ 配置路由
- `/` - 首页 (HomePage)
- `/region/:code` - 结果页 (ResultPage)

### 7. ✅ 3道防线 QA
- [x] **防线1**: ESLint + TypeScript 无错误
- [x] **防线2**: npm run build 成功
- [x] **防线3**: npm run dev 可正常启动，页面可访问

## 技术栈
- React 19 + TypeScript
- Vite 7 (构建工具)
- Tailwind CSS 4 (样式)
- React Router 7 (路由)
- Radix UI (无障碍组件基座)
