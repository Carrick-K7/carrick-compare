# T-019: 清理估算数据，重新收集官方来源GDP数据

## 目标
清理 Compare 项目中所有非官方来源的估算数据，重新收集有明确官方来源的真实数据。

## 问题现状
- 1,478 条记录中有 194 条是**整数值估算数据**
- 主要来自 T-016 县级市历史数据（2015-2023）
- 数据来源为 SubAgent 根据增长率**推算**，非真实官方数据

## 整改方案：严格模式

### 1. 数据清理（先删除）
删除所有**整数值**的县级市历史数据，只保留：
- ✅ 2024 年有官方来源的数据
- ✅ 2025 年有官方来源的数据
- ❌ 删除所有 2015-2023 年的估算整数数据

### 2. 重新收集（后补充）
逐个搜索每个县级市每年的官方 GDP 数据：
- 搜索关键词："昆山市 2020年 GDP"、"江阴 2019 统计公报" 等
- 来源要求：统计局官网、政府工作报告、官方媒体
- 记录来源：每个数据点必须标注来源URL

### 3. 数据来源标注
新增 `source` 字段：
```json
{
  "regionCode": "320583",
  "year": 2024,
  "value": 5380.2,
  "source": "https://www.kunshan.gov.cn/...",
  "sourceName": "昆山市统计局2024年公报"
}
```

## 数据来源标准

| 优先级 | 来源类型 | 示例 |
|:---:|:---|:---|
| 1 | 统计局官网 | 昆山市统计局网站 |
| 2 | 政府工作报告 | XX市2023年政府工作报告 |
| 3 | 统计公报 | XX市2022年国民经济和社会发展统计公报 |
| 4 | 官方媒体 | 人民日报、新华社、第一财经（有具体数字） |
| ❌ | 不接受 | 推算数据、估算数据、没有来源的数据 |

## 风险
- 部分县级市历史数据可能没有公开记录
- 如果找不到某年数据，该年留白，不填充估算值
