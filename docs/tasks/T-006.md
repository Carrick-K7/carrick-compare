# T-006: 查询结果页面布局

## 目标
实现查询结果页面，整合搜索、区域信息、相似推荐、省内排名等模块，形成完整页面。

## 完成状态
✅ 已完成

## 实现内容

### 1. 页面布局
修改 `src/pages/ResultPage.tsx`，实现以下布局：

**桌面端三栏布局** (`lg:grid-cols-12`):
- 左侧栏 (col-span-3): 区域信息卡片 (RegionInfoCard)
- 中间栏 (col-span-6): 历史趋势图表占位 + 其他功能占位
- 右侧栏 (col-span-3): 相似区域推荐占位 + 省内同级排名占位

**移动端单栏堆叠** (`grid-cols-1`):
- 区域信息 → 图表 → 其他功能 → 相似推荐 → 省内排名

### 2. 页面元素
- **顶部**: 返回首页按钮 + 当前区域标题（粘性导航栏）
- **左侧栏**: RegionInfoCard（集成 T-005 组件）
- **中间栏**: 
  - 历史趋势图表占位（预留 T-009）
  - 其他功能占位（预留 T-008）
- **右侧栏**: 
  - 相似区域推荐占位（预留 T-007）
  - 省内同级排名占位（预留 T-008）
- **底部**: 时间粒度切换（年度/半年度），固定底部

### 3. 数据加载
- 从路由参数 `code` 获取 regionCode
- 使用 `getRegionByCode()` 加载对应区域数据
- 使用 `getRegionHistory()` 加载历史GDP数据
- 处理区域不存在的情况（显示 404 页面）

### 4. 修复问题
- 修复了 RegionInfoCard 组件中的 TypeScript 类型错误
- 确保可选参数正确处理 undefined 情况
